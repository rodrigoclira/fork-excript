Nesta aula estudaremos a estrutura de repetição for, a instrução mais utilizada pelos desenvolveres na construção de laços de repetição. Aprenderemos as características e as vantagens dessa estrutura e onde devemos utiliza-la.

LAÇO DE REPETIÇÃO FOR I

A linguagem C fornece algumas estruturas para iterarmos listas ou então, para simplesmente repetirmos um conjunto de instruções por um determinado número de vezes ou enquanto um condição for verdadeira. A estrutura for é especializada em executar um bloco de instruções por um número pré-definido de vezes, por exemplo:

for(int x = 0; x < 10; x++){
    printf(x);
}

No código acima, foi definido que o bloco de instrução da instrução for deve ser repetido enquanto o valor da variável x seja menor do que 10. Logo, temos que o valor de x é incrementado em cada ciclo e isso também é feito no cabeçalho da estrutura for, logo, nós temos que o valor de x é alterado a cada ciclo e em seguida, o mesmo é impresso na tela.
