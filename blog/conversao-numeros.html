TABELA DE CONVERSÃO DE NÚMEROS EM PYTHON

  • ARTIGO

INTRODUÇÃO

Abaixo temos uma tabela onde o objetivo é converte de algum tipo para um tipo numérico ou de um tipo numérico para algum outro tipo. Algumas vezes é necessário importar a função para que seja possível executar o código.

TABELA

DE PARA EXPRESSÃO
45 "45" str(data)
45 "101101" bin(data)
45 "2D" hex(data)
45 b"x00x00x00x2d" struct.pack('!i', data)
"45" 45 int(data)
"45" "3435" data.encode('hex')
"101101" 45 int(data, 2)
"2D" 45 int(data, 16)
"2D" b"x2d" binascii.unhexlify(data) ou data.decode('hex')
"x00x00x00x2d" 45 struct.unpack('!i', data)[0]
"x2d" b"2D" binascii.hexlify(data)
"3435" "45" data.decode('hex')
